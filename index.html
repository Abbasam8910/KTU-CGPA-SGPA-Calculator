How to Calculate

SGPA Calculation

SGPA = Σ(Credit × Grade Point) / ΣCredits

CGPA Calculation

CGPA = Σ(SGPA of each semester) / Number of semesters

Note

For KTU, CGPA is calculated by averaging the SGPA of all semesters completed so far.
